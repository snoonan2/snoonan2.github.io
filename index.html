Projects & Clubs

I worked on a hackathon project that addresses the global education crisis by developing an AI-based solution to mitigate teacher shortages and enhance learning experiences to create an interactive support systems for students in hope to increase retention rates in college.The AI serves as a virtual teacher, providing personalized instruction, interactive lessons, and real-time feedback to students. By leveraging advanced technologies, the project aims to deliver quality education to underserved communities, bridging the digital divide and ensuring equitable access to learning resources. In a comprehensive study conducted in Central Park, New York, researchers collected data on squirrel behavior. In order to draw meaningful conclusions about their activities and interactions within the urban environment, I created a project using scraping the data from the web in python. Through systematic observations and surveys, the project uncovered patterns in squirrel behavior, including daily activity rhythms and habitat preferences. The findings were presented on a dedicated website, featuring interactive visualizations, aimed at promoting awareness and appreciation for urban wildlife among the public.

Utilizing Marvel App software and gathering feedback from students, I developed a prototype application aimed at assisting commuting students with school bus tracking and pick-up time estimation. Partnering with the New York Public Transportation System, the project sought to address issues identified by both students and teachers regarding school bus tracking technology. Through user feedback sessions and collaboration with transportation authorities, the prototype was designed to provide real-time bus tracking, accurate pick-up time estimations, and notifications to improve the commuting experience for students and teachers alike. The application's features were tailored to address specific challenges faced by commuters in navigating the New York City transportation system, fostering efficiency and reliability in school bus services.

This project involved the development of a game in C programming language where users can submit words to generate a crossword puzzle along with hints and an answer key. The program was designed to automate the process of crossword puzzle creation, providing an interactive and engaging experience for players. Through user input, the game dynamically constructs crossword grids, generates corresponding hints for each word, and generates an answer key for reference. The project aimed to offer a fun and educational tool for users to enjoy creating and solving crossword puzzles efficiently using computer algorithms.

This Python project replicates the popular game Wordle, where players attempt to guess a randomized hidden word within a limited number of attempts. The program allows users to input their guesses, providing feedback on the correctness of each guess and indicating the letters that are correctly placed, misplaced, or not present in the target word. Through iterative guessing, players aim to deduce the target word within the specified constraints. The project provides a fun and challenging experience for players, encouraging problem-solving and vocabulary skills development in an interactive setting.
